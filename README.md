# AIIJC - Final Solution
Repository For Accessing Final Solution For Team Round Of AIIJC Contest.
Team: CSPDarknet53
Track: AI in geoservices
Task: Develop a Road Sign Classifier (Dataset provided by 2GIS)


## Solution
- Solution Source Code: Files For Training And Testing Files Source Code
- Please, add class2label.json file to folder where checkpoint will be
- Notebook for final is named EfficientNetB2Final.ipynb


## Approach:
- Developed a multiclass-multilabel image classification model to categorize road signs across 6 basic headings utilizing an EfficientNetB2 architecture.
- Implemented a comprehensive data augmentation pipeline (including optical distortion, color jitter, random shadow, and blur) to enhance model generalization and robustness.
